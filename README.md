## ✨ Features

- 🔑 Geração de códigos **2FA (TOTP)** via Discord  
- ⚡ Comando slash (`/gerar2fa`)  
- 🔒 Respostas privadas (ephemeral)  
- 📦 Leve e rápido  
- 🧠 Compatível com Google Authenticator, Authy, etc
